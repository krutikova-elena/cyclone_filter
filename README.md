# Циклонный фильтр для очистки воздуха ДВС

Проект посвящён разработке и экспериментальному исследованию **циклонного фильтра** для системы очистки воздуха двигателя внутреннего сгорания.

В работе рассмотрены основные физические методы фильтрации воздуха, проведён анализ размеров частиц загрязнений, выполнены расчёты геометрических параметров циклона, перепада давления и эффективности улавливания частиц. На основе расчётов собран и испытан **прототип циклонного фильтра типа ЦН-15**.

Экспериментальная установка включает вытяжной вентилятор, микроконтроллер **Arduino Uno** для измерения оборотов, а также систему сбора и анализа частиц. В ходе экспериментов подтверждена эффективность фильтрации на уровне **≈80–85%** для частиц среднего размера.

Проект выполнен обучающимися 8 класса в рамках инженерно-исследовательского кейса.

## Основные результаты
- Анализ размеров частиц загрязнений (ImageJ, Excel)
- Расчёт параметров циклона и потерь давления
- Сборка прототипа и экспериментального стенда
- Экспериментальная оценка эффективности фильтрации
- Использование Arduino для измерения оборотов вентилятора
